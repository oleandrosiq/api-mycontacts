## ✅ Starting ##

```bash
# Clone this project
$ git clone https://github.com/oleandrosiq/api-contacts

# Access
$ cd api-contacts

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
